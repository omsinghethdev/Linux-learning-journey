1.Kepping your system uptodate
In different linux distribution have different process
but in ubantu we use 'apt' for kepping our system uptodated
-sudo apt update :this gives whole list of updation which packege is to update or which dependencies is needed
-sudo apt upgrade :this runs small update for our system small means it update all the avilable packages which is need to be updated .
-sudo apt full-upgrade or apt dis-upgrade it allows for large updates for the system , large upgrade means upgrade existing packages and removes/installs additional packages(dependencies).


2.Installing and removing software
-apt install [packagename] for installing softwares 
-apt uninstall [packagename] for uninstalling software 
if you are a regular user than you need use 'sudo ' to elevating  the privileges.
-apt autoremove which removes packeges which are no longer needed.
