#Things learned today
File management in bash-'touch', 'mkdir','mv','cp','rm' and 'rmdir'.
touch -used to creat empty file (multiple file)
mkdir(make directory) -this is used to make a new directory.
mv(move)-used to move an exiisting file to another location.
	-can also be used to rename an existing file.
cp(copy)-to copy an existing file
	-also used to rename the file
to copy a whole folder we use 'cp -R'
rm(remove)-to remove file (multiple file at once)
	-for deleting a directory ,use the option -r
rmdir(remove directory)
	-to delete an empty directory
